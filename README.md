# Gaurav Rai — Portfolio

A professional portfolio site built with **HTML**, **CSS**, and **JavaScript**.

## Sections

- **Hero** — Introduction (Gaurav Rai, 3rd year B.Tech CSE, DSA & development)
- **About Me** — Background and focus areas
- **Skills & Technologies** — Programming, DSA, Web Dev, Tools
- **Projects** — Featured projects with links
- **Certificates & Courses** — Completed certifications
- **Experience** — Timeline of roles and education
- **Blogs** — Blog post links
- **Contact** — Email, LinkedIn, GitHub, Twitter + **Download Resume (PDF)** button

## Resume (PDF)

The **Download Resume (PDF)** buttons link to `resume.pdf` in this folder.

**To enable the download:**  
Place your resume file in the project root and name it **`resume.pdf`**.

```
gaurav-portfolio/
├── index.html
├── styles.css
├── script.js
├── resume.pdf   ← Add your PDF here
└── README.md
```

## Run locally

1. Add your `resume.pdf` (optional).
2. Open `index.html` in a browser, or use a local server:

   ```bash
   # Python
   python -m http.server 8000

   # Node (npx)
   npx serve
   ```

3. Update **Contact** links (email, LinkedIn, GitHub, Twitter) and project/blog URLs in `index.html` to your real profiles.

## Customize

- **Content:** Edit `index.html` for text, project titles, descriptions, and links.
- **Styling:** Adjust colors and layout in `styles.css` (see `:root` variables).
- **Resume:** Replace or rename `resume.pdf` as needed; keep the filename `resume.pdf` so the existing buttons work.
